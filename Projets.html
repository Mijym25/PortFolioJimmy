WinDev

J'ai principalement coder en Windev

Ce langage à la possibilité d'être en français, ce qui peut être plus simple à comprendre

Mais le Windev, l'application, permet aussi de créer des fenêtres

Mais il peut aussi facilement éxecuter du SQL modifié, ça ce nomme HFSQL (Hyper File SQL)

HFSQL

HFSQL est une base de données SQL intégrée aux environnements WINDEV, WINDEV Mobile et WEBDEV

HFSQL existe en version Locale, Réseau, Cluster, Client/Serveur (Windows et Linux), Cloud et Mobile

HFSQL offre un format de stockage binairement compatible entre toutes les plateformes

Le HFSQL peut se faire executer sur les applicatifs WinDev

Etat Windev

Les Etats Windev sont des formats de documents customisable que l'on créer

On peut coder une fenêtre pour récupérer les informations, ou même les .ini

Ensuite les etats peuvent comporter des sous-etats, ce sont des etat imprimer dans d'autre etat

Un etat peut comporter des blocs que l'on modifie avec du code
